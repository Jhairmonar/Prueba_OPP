📚 Fundamentos de Programación

Variables

Espacios reservador en memoria con un tipo de dato específico (int, double, string).

Entrada/Salida

Uso de flujos (cin/cout) para interactuar con el usuario y mostrar resultados.

Operadores

Símbolos que ejecutan operaciones matemáticas y lógicas (sumas, comparaciones).

Condicionales

Estructuras (if/else) que permiten la toma de decisiones según una condición.

Ciclos

Repeticiones controladas (for, while) para ejecutar bloques de código múltiples veces.

Arreglos

Colecciones de datos del mismo tipo almacenados de forma secuencial.

Persistencia de Archivos

Almacenamiento permanente de información en archivos .txt mediante flujos de salida (ofstream).
