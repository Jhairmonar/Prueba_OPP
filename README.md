# Software Academy - Examen Unidad 2 🍎

Este repositorio contiene el desarrollo de la **Prueba Práctica de la Unidad 2** para la asignatura de **Algoritmos y Lógica de Programación**. El proyecto consiste en un aplicativo interactivo que integra un motor de cálculo en **C++** con una interfaz web tipo **Dashboard** profesional.

## 🛠️ Tecnologías Utilizadas
* **C++:** Lógica de programación, arreglos, condicionales y persistencia.
* **HTML5 & CSS3:** Interfaz de usuario con diseño Dark Mode y barra lateral.
* **GitHub:** Control de versiones y gestión de repositorio.
